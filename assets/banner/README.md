# Bannière officielle
## VestigiumLabs

Version : 1.0

---

# Objectif

La bannière officielle représente l'identité visuelle de VestigiumLabs.

Elle est utilisée sur les supports nécessitant une présentation large du laboratoire.

La bannière est une ressource officielle.

Elle ne doit pas être modifiée.

---

# Versions officielles

Le dossier contient les versions validées de la bannière.

Aucune autre variante ne doit être utilisée sans validation.

---

# Organisation

```
banner/

banner.png

banner.svg
```

Les noms des fichiers doivent rester stables afin de faciliter leur utilisation dans les différents projets.

---

# Utilisation

La bannière peut notamment être utilisée :

- sur la page d'accueil du site ;
- comme couverture GitHub ;
- dans la documentation ;
- dans les présentations du laboratoire.

---

# Règles

Conserver les proportions.

Conserver les couleurs officielles.

Ne pas modifier la composition.

Ne pas ajouter de texte.

Ne pas recadrer la bannière de manière à masquer des éléments importants.

---

# Arrière-plan

La bannière est conçue comme une composition complète.

Elle ne doit pas être utilisée comme texture, motif ou fond répétitif.

---

# Taille

La bannière peut être redimensionnée de manière proportionnelle.

Toute déformation est interdite.

---

# Évolution

Toute nouvelle déclinaison de la bannière doit être validée avant son intégration dans ce dossier.
