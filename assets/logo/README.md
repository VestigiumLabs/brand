# Logo officiel
## VestigiumLabs

Version : 1.0

---

# Objectif

Le logo officiel constitue l'identité visuelle principale de VestigiumLabs.

Il permet d'identifier immédiatement le laboratoire sur l'ensemble de ses supports.

Le logo est une ressource officielle.

Il ne doit pas être modifié.

---

# Versions officielles

Le dossier contient les versions validées du logo.

Aucune autre variante ne doit être utilisée sans validation.

---

# Organisation

```
logo/

logo.svg

logo.png

logo-dark.png

logo-light.png
```

Les noms de fichiers doivent rester stables afin de faciliter leur utilisation dans les différents projets.

---

# Utilisation

Le logo peut être utilisé notamment :

- sur le site officiel ;
- dans la documentation ;
- sur GitHub ;
- dans les présentations du laboratoire.

---

# Règles

Conserver les proportions.

Conserver les couleurs officielles.

Respecter la zone de protection définie dans le Design System.

Ne pas appliquer d'effet supplémentaire.

Ne pas modifier la composition.

---

# Arrière-plan

Le logo doit être placé sur un fond offrant un contraste suffisant.

Lorsque plusieurs versions du logo existent, utiliser celle qui garantit la meilleure lisibilité.

---

# Taille

Le logo doit rester parfaitement lisible.

Si une taille très réduite est nécessaire, une version dédiée pourra être créée ultérieurement.

---

# Évolution

Toute nouvelle déclinaison du logo doit être validée avant son intégration dans ce dossier.
