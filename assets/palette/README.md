# Palette officielle
## VestigiumLabs

Version : 1.0

---

# Objectif

La palette officielle définit les couleurs autorisées dans les interfaces de VestigiumLabs.

Chaque couleur possède une fonction.

Les couleurs ne doivent jamais être choisies uniquement pour des raisons esthétiques.

---

# Principe

Une couleur = une responsabilité.

Le même rôle doit toujours utiliser la même couleur.

---

# Palette principale

| Nom | HEX | Utilisation |
|------|-----|-------------|
| Background | #05080D | Fond principal |
| Surface | #0C1522 | Cartes et panneaux |
| Surface Secondary | #20344F | Éléments secondaires |
| Primary | #4F86D9 | Actions principales |
| Text Primary | #F5F7FA | Texte principal |
| Text Secondary | #C7CCD4 | Texte secondaire |

---

# Couleurs fonctionnelles

| Nom | HEX | Utilisation |
|------|-----|-------------|
| Success | #3FA66B | Confirmation |
| Warning | #D9A441 | Avertissement |
| Danger | #D95C5C | Erreur ou suppression |

---

# Utilisation

## Background

Utilisé uniquement comme fond principal.

---

## Surface

Utilisé pour :

- cartes
- panneaux
- navigation
- footer

---

## Surface Secondary

Utilisé lorsque plusieurs niveaux de profondeur sont nécessaires.

Ne jamais remplacer Surface sans justification.

---

## Primary

Utilisé pour :

- boutons principaux
- liens
- focus
- éléments interactifs

Une seule couleur d'action est utilisée dans l'ensemble du Design System.

---

## Text Primary

Utilisé pour tous les contenus principaux.

---

## Text Secondary

Utilisé pour :

- descriptions
- métadonnées
- informations complémentaires

---

# Règles

Ne jamais utiliser une nouvelle couleur sans validation.

Ne jamais utiliser une couleur uniquement pour décorer une interface.

Chaque couleur doit conserver la même signification dans tout le laboratoire.

---

# Accessibilité

Les contrastes doivent respecter les recommandations WCAG.

Les couleurs ne doivent jamais être le seul moyen de transmettre une information.

Les états critiques doivent toujours être accompagnés d'un texte ou d'une icône.

---

# Évolution

Toute nouvelle couleur doit répondre à un besoin concret.

La palette officielle reste volontairement limitée afin de préserver la cohérence visuelle.
