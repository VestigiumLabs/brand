# VestigiumLabs Brand

> Identité visuelle officielle de VestigiumLabs.

---

# Présentation

Ce dépôt constitue la référence officielle de l'identité graphique de VestigiumLabs.

Il centralise l'ensemble des ressources nécessaires à la conception d'interfaces cohérentes pour les différents projets du laboratoire.

Le Design System documenté ici garantit une identité homogène, durable et facilement maintenable.

---

# Objectifs

Le dépôt poursuit plusieurs objectifs :

- garantir une identité visuelle cohérente ;
- faciliter le travail des Designers ;
- fournir un référentiel unique au Developer ;
- assurer la pérennité des choix graphiques ;
- éviter les divergences entre les projets.

---

# Principes

Le Design System suit les principes fondateurs de VestigiumLabs.

## Simplicité

Chaque élément graphique possède une utilité.

Aucun effet décoratif inutile.

---

## Lisibilité

Les contenus restent prioritaires.

Le design met en valeur l'information sans la masquer.

---

## Cohérence

Tous les composants reposent sur les mêmes règles :

- couleurs
- typographies
- espacements
- composants
- interactions

---

## Accessibilité

Les interfaces doivent rester lisibles et utilisables par le plus grand nombre.

Les contrastes et les tailles de texte respectent les bonnes pratiques WCAG.

---

## Pérennité

Les choix graphiques privilégient la stabilité plutôt que les tendances.

Le système doit pouvoir évoluer progressivement sans rupture visuelle.

# Gouvernance

Le logo officiel ne peut être modifié.

La bannière officielle ne peut être modifiée.

Toute évolution du Design System doit être documentée.

Les décisions importantes doivent être validées par le directeur du laboratoire.

# Compatibilité

Le Design System est conçu pour être utilisé avec :

- HTML5
- CSS3
- TailwindCSS
- React
- Next.js

sans dépendre d'une technologie particulière.

---

# Philosophie

Le Design System ne cherche pas à produire une interface spectaculaire.

Il cherche à produire une interface :

- claire ;
- professionnelle ;
- robuste ;
- durable.

L'identité de VestigiumLabs repose sur la rigueur plutôt que sur les effets visuels.
